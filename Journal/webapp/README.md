# Journal Web App

A modern web application for journaling, habit tracking, and creative expression.

## Features

- **Today Screen**: Daily summary with habit tracking and journal entry
- **Calendar Screen**: Month/Week/Day views with events and journal entries
- **Creative Journal**: Canvas-based journal with drawing, stickers, photos, and text
- **Profile Screen**: Manage habits and settings

## Getting Started

### Prerequisites

- Node.js 18+
- Backend API running on port 8000

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at http://localhost:3000

### Build

```bash
npm run build
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Zustand (state management)
- Fabric.js (canvas drawing)
- date-fns (date utilities)

## API Integration

The web app connects to the backend API at `http://localhost:8000/api/v1`. Make sure the backend is running before starting the web app.
